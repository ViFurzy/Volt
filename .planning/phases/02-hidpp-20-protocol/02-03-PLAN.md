The protocol is a single direct command — NOT a HID++ 2.0 feature discovery chain:
  Write:    [0x11, 0xFF, 0x06, 0x0D] + 16 zero-padding bytes (20 bytes total)
  Response: [0x11, 0xFF, 0x06, 0x0D, V_hi, V_lo, state, ...]
    response[4:6] = voltage mV big-endian
    response[6]   = 0x01 discharging, 0x03 charging

Voltage is converted to percentage using the G Pro X calibration curve from HeadsetControl
(piecewise linear interpolation between 6 reference points).

<!-- Hardware-confirmed protocol from 02-01-SUMMARY.md -->

Battery write (20 bytes):
  [0x11, device_idx, 0x06, 0x0D, 0x00 × 16]
  Where device_idx = 0xFF for G Pro X Wireless

Battery response (20 bytes):
  [0x11, 0xFF, 0x06, 0x0D, V_hi, V_lo, state, 0x00, ...]
  response[4] = V_hi   (voltage high byte)
  response[5] = V_lo   (voltage low byte)
  response[6] = state  (0x01 = discharging, 0x03 = charging)

OFFSET = 1: response[0] IS the report ID (0x11). Data starts at index 1.
Live example: [0x11, 0xFF, 0x06, 0x0D, 0x0D, 0xF4, 0x01, ...] → 3572 mV, discharging

G Pro X calibration curve (from HeadsetControl reference):
  Percent: [100,  50,  30,  20,   5,   0]
  mV:      [4150, 3830, 3780, 3740, 3670, 3320]
  (Use piecewise linear interpolation between adjacent reference points)

    Module docstring: explain G-series headset battery protocol for G Pro X Wireless.
    Note: "# Protocol confirmed in 02-01 hardware probe: 0xFF43 interface, device_idx=0xFF"

    Imports:
      from dataclasses import dataclass
      from hidpp.protocol import send_and_recv

    Constants:
      _CALIB = [(3320, 0), (3670, 5), (3740, 20), (3780, 30), (3830, 50), (4150, 100)]
      # (mV, percent) calibration points for G Pro X Wireless from HeadsetControl reference

    @dataclass class BatteryResult:
      percent: int
      charging: bool
      feature_used: str

    Function voltage_to_percent(mv: int) -> int:
      if mv <= _CALIB[0][0]: return 0
      if mv >= _CALIB[-1][0]: return 100
      for i in range(len(_CALIB) - 1):
          mv_lo, pct_lo = _CALIB[i]
          mv_hi, pct_hi = _CALIB[i + 1]
          if mv_lo <= mv <= mv_hi:
              frac = (mv - mv_lo) / (mv_hi - mv_lo)
              return round(pct_lo + frac * (pct_hi - pct_lo))
      return 0

    Function battery_probe_chain(device, device_idx: int) -> BatteryResult | None:
      Docstring: "Read battery from G Pro X Wireless via direct G-series headset command
      (feature 0x06, function 0x0D). Returns None on timeout (device offline or off).
      device_idx should be 0xFF (receiver device index for G-series headsets)."

      cmd = [0x11, device_idx, 0x06, 0x0D] + [0x00] * 16
      result = send_and_recv(device, cmd, min_len=7)
      if result is None:
          return None
      voltage_mv = (result[4] << 8) | result[5]
      charging = result[6] == 0x03
      return BatteryResult(
          percent=voltage_to_percent(voltage_mv),
          charging=charging,
          feature_used="0x06/0x0D",
      )

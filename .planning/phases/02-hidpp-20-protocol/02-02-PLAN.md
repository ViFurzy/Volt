    - open_receiver(info: dict) -> hid.device: calls hid.device(), then open_path(info["path"]), returns the open device object (does NOT close — caller owns lifetime)
    - discover_device_index(device) -> int | None:
        Probes indices 0x01, 0x02, 0x03, 0x04, 0x05, 0x06 in order.
        For each index: calls build_short_msg(device_idx=idx, feature_idx=0x00, function=0) to build
        a Root feature query, then calls send_and_recv(device, msg, timeout_ms=100).
        If result is not None and result is a non-error response: returns idx.
        On HIDppError: continues to next index (error at one index does not stop probing).
        On OSError: continues to next index.
        If no index responds: returns None.
    - 0xFF is NEVER passed as device_idx in any call inside receiver.py
    - discover_device_index returns None (not raise) when mock returns [] for all indices
    - discover_device_index returns 0x01 when mock returns a valid Root response for index 0x01
    - discover_device_index skips an index that raises HIDppError and returns the next valid index

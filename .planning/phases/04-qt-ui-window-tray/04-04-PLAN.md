<interfaces>
<!-- The full assembled pipeline under test -->
__main__.py: QApplication -> setQuitOnLastWindowClosed(False) -> setStyleSheet(DARK_QSS)
           -> MainWindow -> TrayManager(window,qapp).show()
           -> MonitorApp(consumer=window.on_device_update).start()
           -> build_hotplug() / make_timer() -> window.show() -> qapp.exec()

MonitorApp.drain() pops DeviceState from ui_queue and calls consumer(state) on the main thread.
MainWindow.on_device_update(state) create-or-updates a DeviceCard keyed by (vid,pid,dev_idx).
</interfaces>
